# Legacy Workspace

Esta pasta concentra os blocos legados e materiais antigos que nao fazem parte do caminho principal de execucao.

## Caminho principal

Os diretórios ativos do projeto ficam em:

- `apps/api`
- `apps/web`
- `apps/ai-service`
- `apps/desktop`
- `packages/*`
- `modules/*`
- `WHATSAPP`
- `MEDIA`

## Conteudo preservado

- `legacy/FRONTEND`
- `legacy/IA`
- `legacy/RH`
- `legacy/FINANCEIRO`
- `legacy/CONTABILIDADE`
- `legacy/INFRA`

Essas pastas continuam no repositório para consulta, migração e reaproveitamento seletivo, mas nao sao a superfície principal do produto.
