# Innovation RH Connect

Sistema de RH com controle de ponto e comunicação via WhatsApp para pequenas e médias empresas.

## 🚀 MVP Scope

Este projeto foca na entrega de um protótipo vendável focado em Gestão de RH e Comunicação Corporativa.

### Módulos Principais
- **Dashboard RH**: Indicadores de funcionários, ponto e mensagens.
- **Gestão de Funcionários**: Cadastro completo, cargos e departamentos.
- **Controle de Ponto**: Registro e espelho de ponto, banco de horas.
- **Férias**: Solicitação e aprovação de períodos de descanso.
- **WhatsApp Corporativo**: Comunicação integrada via QR Code para avisos e convocações.

## 🛠 Tech Stack

- **Frontend**: Next.js 14, TailwindCSS, Lucide, Recharts, Framer Motion.
- **Backend**: NestJS, Prisma, PostgreSQL, JWT, Socket.io, Baileys (WhatsApp).
- **Security**: bcrypt, Throttler, Helmet, Audit Logs.

## 📁 Estrutura do Projeto

- `apps/web`: Frontend Next.js.
- `apps/api`: Backend NestJS.
- `apps/ai-service`: Serviço de IA (Oculto no MVP).
