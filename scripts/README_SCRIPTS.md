# 📋 Scripts & Automation

This directory contains shell scripts, batch files, and automation tools for the Innovation IA project.

## Windows Scripts (.bat)

### INICIAR_INNOVATION.bat
Main startup script for the entire Innovation IA ecosystem.
```bash
.\scripts\INICIAR_INNOVATION.bat
```

### iniciar_app_desktop.bat
Starts the Electron desktop application.
```bash
.\scripts\iniciar_app_desktop.bat
```

### TESTAR_APP.bat
Runs the complete test suite for all applications.
```bash
.\scripts\TESTAR_APP.bat
```

## PowerShell Scripts (.ps1)

Located in this directory, these scripts provide advanced automation:
- `deploy_vps.ps1` - Deploy to VPS
- `run_local.ps1` - Run locally
- `run_backend.ps1` - Start backend only
- `run_local_debug.ps1` - Debug mode

## Usage

Run any batch file directly:
```bash
cd scripts
.\INICIAR_INNOVATION.bat
```

Or from the root:
```bash
.\scripts\INICIAR_INNOVATION.bat
```
